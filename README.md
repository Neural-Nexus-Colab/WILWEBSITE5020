# Empowering the Nation - Course Platform

## Project Overview

Empowering the Nation is a web-based learning platform offering skills development through two program types:
- **6-week short courses** - Focused, practical skill-building programs
- **6-month comprehensive programs** - In-depth professional certification courses

The platform provides course information, fee calculation, registration, and user account management.

## Site Structure

### Main Pages

1. **Home** - Landing page with course offerings, calculator access, success stories, and call-to-action buttons
2. **About** - Organization mission, values, history, impact metrics, and testimonials
3. **6-Week Courses** - Short course catalog with hover-to-view details
4. **6-Month Courses** - Professional program catalog with hover-to-view details and FAQ section
5. **Contact Us** - Contact form, FAQ section, and social media links
6. **Register & Calculator** - Course selection, fee calculator with discount breakdown, and registration form
7. **Login** - User authentication with social login options
8. **Sign Up** - New user registration with social sign-up options

---

## Key Features

### Course Browsing
- Hover interaction reveals detailed course information (description, benefits, pricing)
- Direct navigation to registration from course pages

### Fee Calculator
- Real-time cost calculation based on course selection
- Automatic discount application for multiple courses
- VAT calculation and total breakdown
- Clear display of subtotal, discounts, and final amount

### Registration System
- Multi-course selection
- Contact information collection (name, email, phone, ID, card number)
- Submit and clear functionality

### User Accounts
- Traditional email/password authentication
- Social login options (Google, Twitter, Facebook)
- Separate login and sign-up flows
